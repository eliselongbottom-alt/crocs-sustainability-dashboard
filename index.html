How to Get Live Trending Data — Bypassing Google's Block

Google Trends has no official API. Three ways to pull real trending search data.

Recommended
SerpAPI (free tier)
Reliable, never blocked. Free tier = 100 searches/month — enough for weekly refreshes across all markets.
  1. Sign up at serpapi.com — free, no credit card
  2. Copy your API key from the dashboard
  3. Run: python3 fetch_trends.py --api serpapi --key YOUR_KEY
  4. Drag the generated Excel onto the upload zone above
Free / Unstable
pytrends + Chrome cookies
Free Python library. Adding your Chrome cookies dramatically reduces Google's 429 blocking.
  1. Install: pip install pytrends browser-cookie3
  2. Run: python3 fetch_trends.py --cookies
  3. If blocked: try a different network or wait a few hours
Manual
Export directly from Google Trends
Zero code. Takes ~15 min. Always works.
  1. Go to trends.google.com → Trending Now
  2. Select each country, download the CSV
  3. Paste into the Excel template (one sheet per market)
  4. Drop the completed file above
